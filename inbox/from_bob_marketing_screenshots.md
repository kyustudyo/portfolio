# [from: 밥먹어쒀팀] 앱스토어 마케팅 스크린샷 — 공통 규격 안내

카카오톡 스토어 페이지처럼 **배경 + 문구 + 기기 프레임**이 있는 '예쁜' 마케팅 스크린샷을
모든 프로젝트에 동일한 방식으로 제공하려 합니다. 아래 규격·폴더 규칙·재사용 스크립트를 참고하세요.

## 1. 비율 / 사이즈 (세로, 리사이즈 금지 — 스토어가 정확 픽셀 요구)
| 스토어/기기 | 픽셀 | 비율 |
|---|---|---|
| iOS App Store 6.7"(권장) | **1290 × 2796** | ≈ 9 : 19.5 |
| iOS App Store 6.5" | 1284 × 2778 | ≈ 9 : 19.5 |
| iPad 12.9" | 2048 × 2732 | 3 : 4 |
| Android Play (폰) | **1080 × 1920** (~1080×2340까지) | 9 : 16 |
| Android 7" 태블릿 | 1200 × 1920 | |
| Android 10" 태블릿 | 1600 × 2560 | |

- **원본 스크린샷은 각 스토어의 '네이티브 해상도'로 캡처**하고, 마케팅 합성본도 **같은 사이즈로 출력**하세요. (작게 찍어 늘리면 흐려지고 심사 반려)
- 앱스토어 최소 개수: 스크린샷 3~10장. 보통 4~5장이면 충분.

## 2. 어디에 둘까 (폴더 규칙 — 모든 프로젝트 동일)
```
store-assets/screenshots/
  ios/            ← 원본(01_xxx.png, 02_xxx.png ...)
  android/  ipad/  android-7inch/  android-10inch/   ← 각 사이즈 원본
  ios-marketing/  ← 합성(예쁜) 결과. 파일명은 원본과 동일
```
- 파일명은 `01_today.png`, `02_calendar.png` 처럼 **번호_화면이름**으로 통일.
- File Browser로 다운로드: `http://100.99.196.12:8889/file_browser.html#{프로젝트}/screenshots/ios-marketing`
  (Tailscale 켜져 있어야 접속)

## 3. 디자인 스펙 (통일감)
- 배경: 브랜드색 그라데이션(위→아래, 밝게). 밥먹어쒀는 핑크(#FFE0E7)→크림(#FFF5F0).
- 상단: 브랜드칩(앱아이콘 + 앱이름, 흰 알약형) → 그 아래 **2줄 헤드라인**(딥 브랜드색, Bold ~98px) → **서브라인**(회색 ~44px).
- 기기 카드: 원본 스크린샷을 **둥근 모서리(radius ~84) + 흰 베젤(10px) + 부드러운 그림자**로. 폭은 캔버스의 ~72%.
- 폰트: 한글은 AppleSDGothicNeo Bold(맥 기본). PIL로 `ImageFont.truetype('/System/Library/Fonts/AppleSDGothicNeo.ttc', size, index=<Bold>)`.

## 4. 재사용 스크립트
밥먹어쒀가 만든 합성 스크립트를 그대로 복사해서 **문구(specs)·배경색·폰트 크기만** 바꾸면 됩니다:
- 경로: `/Users/hankyulee/projects/bob/store-assets/make_marketing_screens.py`
- 필요: `pip3 install pillow` (PIL). 맥이면 한글 폰트 기본 제공.
- specs 리스트에서 각 화면의 `[헤드라인 2줄], '서브라인'`만 프로젝트에 맞게 수정 → `python3 make_marketing_screens.py`.

프로젝트 톤(색·문구)에 맞춰 세팅해 드릴 수 있습니다. 필요하면 회신 주세요.
