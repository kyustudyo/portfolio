채용 공고를 분석해 맞춤형 면접 퀴즈를 자동 생성하는 도구. 회사별 퀴즈 누적, 난이도별 문제 출제, 코드 하이라이팅, 오답노트 기능 포함. localStorage 기반으로 서버 없이 동작하며, 답변 작성 → 정답·핵심 포인트 확인 → 오답 체크의 학습 사이클을 지원합니다.
